A token example of an exploratory project using R/python.  This was an approach I took for certain early MGI projects I had.

```
├── Makefile
├── bin
│   ├── analysis.r
│   ├── bsub.sh
│   ├── bsub2.sh
│   ├── calculate-build-filter-ratios.pl
│   ├── install-pkgs.r
│   └── run-vcf.sh
├── data
    └── external
    └── derived
├── logs
├── reports
    └── markdown
    └── latex
├── supplemental
│   └── docs
└── vendor
    └── rpkgs
    └── virtualenv
```
